# Security Policy

## Supported Versions

Only the latest release is supported with security fixes.

| Version | Supported |
|---------|-----------|
| latest  | yes       |

## Reporting a Vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](https://github.com/OfBirds/difflab/security/advisories/new).
You will receive an acknowledgement within 72 hours. If a fix is warranted, we will coordinate a
disclosure timeline with you before publishing.
